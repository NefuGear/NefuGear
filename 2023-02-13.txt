2.프로젝트 세팅

맵 @ 모드

디폴트 모드 - 월드 세팅에 게임모드 오버라이드가 'None'일때 - 기본 설정으로 작동

폰 클래스 - (유저가 조종하는) 몸
HUD 클래스 - 화면에 보여지는 UI
플레이어 컨트롤러 클래스 - (유저가 조종할 때의) 컨트롤 하는 수단 
	: 포세스(빙의)개념임 폰 클래스에서 설정된 클래스 존재를 조종하는 클래스
게임 스테이트 클래스 - 게임의 전체적인 설정 (Ex - 축구 팀의 스탯)
플레이어 스테이트 클래스 - 각 개체의 설정 (Ex - 축구 팀 내의 선수의 스탯)
관전자 클래스 - 게임을 관전하는 시스템 설정

디폴트 맵

에디터 시작 맵 - 프로그램 실행시 기본 레벨 설정
게임 기본 맵 - 작업 실행시 기본 게임 설정 맵

3. 블루프린트

블루 프린트는 실행 라인에 연결된 상태에 따라서 실행 순서가 달라진다

이벤트 - 무언가 실행하기 위한 타이밍(EX - 점심시간이 11시 30분일 때) / 액션 - 이벤트가 달성 될 시의 행동(EX - 점심먹으러 나간다.)

	엑터에 대한 이벤트
            엑터에 대한 엑션
            컴포넌트에 대한 이벤트
            컴포넌트에 대한 엑션

변수 

+ : 더하기 / - : 빼기 / * : 곱하기 / '/' - 나누기 / %  : 나머지
>= / == / != / <=

위치 

Set Relative Location - 벡터 값이다.
핀 분할 = 하고싶으면 

루트 컴포넌트 : 모든 액터의 근본이 되는 컴포넌트 설정이 된 것은 좌표를 지정할수가 없다.
뷰포트 시뮬레이션 : 블루프린트 상에서의 시뮬레이션 버튼은 레벨에서 확인하는 시뮬레이션보다 간편하게 시뮬레이션을 확인한다.
